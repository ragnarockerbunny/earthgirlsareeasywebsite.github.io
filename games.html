Project Tree

Project Management has never been easier.

What are we

The project entails the development of a project management system. The system represents a project visually as a tree. The core idea of the system is to allow for all task dependencies to be captured accurately and clearly, to ensure effective project management.

What makes us different

Created by Project Tree as part of UP's COS301 year project, Project tree changes the way people usually go about Project Management. By visualizing the graph's tasks in movable, bubbled nodes, automatic adjusting dependencies between tasks and a clear colour coded key; Project tree is the next step in organization.
